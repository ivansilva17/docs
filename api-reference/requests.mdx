---
title: 'Requests'
description: 'Full Fabric uses HTTP verbs and a RESTful endpoint structure. Request and response payloads are formatted as JSON.'
---

```shell
curl \
  --request GET \
  --url https://_.fullfabric.com/apis/profiles/profiles/ID \
  --header 'Accept: application/json;version=2021-04-13' \
  --header 'FF-Auth: TOKEN'
```

## Common request headers

| Name          | Type                 | Default                                      |
| ------------- | -------------------- | -------------------------------------------- |
| FF-Auth	      | string               | ```[user token]```        |
| Accept        | string               | ```application/json;version=[api version]``` |
